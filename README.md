# Nodinator
[![CircleCI](https://circleci.com/gh/MadeWilson/nodinator.svg?style=svg)](https://circleci.com/gh/MadeWilson/nodinator)
## Build
  ````
  $ ng build
  $ npm run start:dev
  ````
  
## Generate fake API data
Make sure MongoDB is running and run :
 ````
 $ npm run generator:content
 ````

## Running unit tests

  ````
  $ npm run api:test
  ````
